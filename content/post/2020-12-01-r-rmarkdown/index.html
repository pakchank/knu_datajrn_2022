소개

이 블로그는 2022년 경북대학교 데이터 저널리즘 수업 최종 프로젝트를 위해 제작되었습니다. 이곳에 실린 기사들은 수업을 수강하는 학생들이 최선을 다해 작성한 데이터 저널리즘 기사들입니다. 재미있게 읽어주시길 바랍니다.

You can embed an R code chunk like this:

summary(cars)
